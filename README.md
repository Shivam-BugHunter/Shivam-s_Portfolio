# Shivam Mishra | Portfolio

A premium, highly interactive portfolio website built with React, Vite, and Framer Motion.

## 🚀 Live Demo
Check out the live site: [https://neon-chaja-e964ea.netlify.app/](https://neon-chaja-e964ea.netlify.app/)

## 🛠️ Tech Stack
- **Frontend**: React.js
- **Styling**: Vanilla CSS (Custom Design System)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Build Tool**: Vite

## ✨ Features
- Custom interactive cursor
- Glassmorphism UI
- Scroll-reveal animations
- Responsive design
- SEO optimized

## 📦 Installation
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev` to start the development server
